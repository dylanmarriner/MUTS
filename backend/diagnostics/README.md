# Diagnostics Modules

## MDS (Mazda Diagnostic System)

- Engine diagnostics and protocols
- ABS system diagnostics
- SRS/airbag diagnostics
- Body control modules

## MUTS (Mazda Universal Tuning System)

- Main scanner functionality
- DTC reading/clearing
- Live data streaming
- Service functions

## ADD (Auxiliary Diagnostic Drivers)

- Extended sensor support
- PID extensions
- Experimental features
